# App Screenshots

Place your app screenshots in this directory for app store submissions.

## Required Screenshots

### For App Stores

**iOS App Store**:
- 6.5" iPhone (iPhone 14 Pro Max, 13 Pro Max, 12 Pro Max): 1284x2778 or 1242x2688
- 5.5" iPhone (iPhone 8 Plus): 1242x2208
- 3-10 screenshots required per device size

**Google Play Store**:
- Phone: 1080x1920 or 1080x2340
- 2-8 screenshots required

### PWA Manifest Screenshots
- screenshot-dashboard.png (1242x2688)
- screenshot-workorders.png (1242x2688)
- screenshot-chat.png (1242x2688)

## Screenshot Guidelines

**Content to Capture**:
1. Dashboard with AI insights panel showing
2. Work orders list with filters
3. AI chat assistant in action
4. Job creation flow
5. Mobile responsive view

**Design**:
- Clean, professional look
- Show actual app functionality
- Highlight AI features with blue-purple gradients
- Black/white theme should be prominent
- Include realistic data (not placeholder)

**Tips**:
- Use Chrome DevTools device toolbar for consistent sizing
- Take screenshots in "standalone" display mode (after PWA installation)
- Avoid showing sensitive/personal information
- Ensure high contrast and readability
- Add subtle frames or device mockups (optional)
